# Rust Terminal Snake Game

## Run

Use the Bash launcher:

./run_snake.sh

The launcher builds the release binary if needed and starts the game.

## Controls

- Arrow keys: change direction

## Rules

- Eat apples to grow and increase score.
- Collision with wall or snake body ends the game.
- Score is not shown during gameplay.
- Name prompt appears only for a new all-time high score.
